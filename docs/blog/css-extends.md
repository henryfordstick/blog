# css 那些属性可以继承
下面都是常见的 css 可以继承穿透的属性
## 一、字体属性系列
- font-family：字体系列

- font-weight：字体的粗细

- font-size：字体的大小

- font-style：字体的风格

## 二、文本系列属性
- text-indent：文本缩进

- text-align：文本水平对齐

- line-height：行高

- word-spacing：单词之间的间距

- letter-spacing：中文或者字母之间的间距

- text-transform：控制文本大小写（就是 uppercase、lowercase、capitalize 这三个）

- color：文本颜色

## 三、元素可见性
- visibility：控制元素显示隐藏

## 四、列表布局属性
- list-style：列表风格，包括 list-style-type、list-style-image 等

## 五、光标属性
- cursor：光标显示为何种形态
